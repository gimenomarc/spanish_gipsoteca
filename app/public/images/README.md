# Organización de Imágenes

## Estructura de carpetas:

- **`sculptures/`** - Imágenes principales de las esculturas (alta resolución)
- **`hero/`** - Imágenes para la sección hero/portada
- **`thumbnails/`** - Miniaturas de las esculturas (para galerías)

## Cómo usar:

1. Descarga las imágenes desde Google Drive
2. Colócalas en las carpetas correspondientes
3. Nombra los archivos de forma descriptiva (ej: `venus-de-milo.jpg`, `moises.jpg`)
4. El código se actualizará automáticamente para usar estas imágenes

## Formatos recomendados:
- JPG o PNG
- Resolución: 1200x1600px o superior para imágenes principales
- Resolución: 400x600px para thumbnails
